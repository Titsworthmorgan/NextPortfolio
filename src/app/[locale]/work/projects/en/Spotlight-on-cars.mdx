## Overview

I worked part-time as a developer for Spotlight on Cars, a project aimed at creating an enthusiast vehicle-focused community, marketplace, and entertainment platform. From the start we focused on building a robust and scalable platform using modern technologies to deliver a seamless user experience.

## Key Features

- **Architecture and Development**: Contributed to the architecture and development of the platform using Next.js for the frontend and Neon for the backend, ensuring a responsive and dynamic user interface.
- **Database Management**: Utilized PostgreSQL for efficient data storage and retrieval, ensuring the platform could handle large volumes of user data and transactions.
- **Cloud Infrastructure**: Leveraged Vercel for CI/CD and Google Cloud Platform (GCP) for scalable cloud infrastructure, ensuring high availability and performance.

## Technologies Used

- **Next.js**: For building a server-rendered, performant web application.
- **Neon**: As a serverless Postgres database for scalable and efficient data management
- **PostgreSQL**: For robust relational database management.
- **Vercel**: For continuous integration and deployment, ensuring smooth updates and releases.
- **Google Cloud Platform (GCP)**: For scalable cloud infrastructure and services.

## Challenges and Learnings

From inception to launch, the project presented several challenges, including designing a scalable architecture that could accommodate future growth and ensuring seamless integration between various technologies. Collaborating with a cross-functional team required effective communication and coordination to align on project goals and timelines. Additionally, optimizing performance and user experience was a continuous learning process, involving regular testing and iteration.

## Outcome

The Spotlight on Cars platform was successfully launched with a range of features tailored to automotive enthusiasts. The development team not only gained valuable experience in building a complex web application but also contributed to a project that fosters community engagement and knowledge sharing among car lovers.
